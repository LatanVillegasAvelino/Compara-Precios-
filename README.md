# Compara Precios Perú

Proyecto de comparador de precios para tiendas de Perú.  
Sitio estático publicado con **GitHub Pages**.

## URL pública

https://LatanVillegasAvelino.github.io/Compara-Precios-/

## Características

- Búsqueda de productos (demo).
- Tarjetas de resultados con rangos de precios.
- Sección de tiendas destacadas.
- Exploración por regiones del Perú.
- SEO básico (meta tags, Open Graph, Twitter Card).
- Sitemap, robots.txt, manifest.json y security.txt configurados.

```text
